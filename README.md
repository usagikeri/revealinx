# Using
1. create markdown file 
2. `$cp slide.md ~/.slide/slide.md`
3. `$cp markdown-file ~/.slide/slide.md` 

---

or

1. chmod a+x slide
2. cp slide /usr/local/bin/
3. slide

---

# slide command
e.g.
1. slide markdown-file
copy markdown file
2. slide -o markdown-file
copy markdown file and open localhost:8080
3. slide -o
open localhost:8080
4. slide -h
show help
5. slide
show help
